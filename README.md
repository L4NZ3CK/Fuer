# Fuer
Con esta herramienta podras hacer ataques de fuerza bruta a diferentes cosas.

Necesario ser usuario root o usar kali linux en root

INSTALACION
git clone https://github.com/L4NZ3CK/Fuer
cd Fuer
chmod +x *
bash Fuer.sh
